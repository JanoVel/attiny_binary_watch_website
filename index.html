Description

This site describes how a binary watch was made using an Arduino programmed ATTINY and a DS1302, source code is on GitHub.

It’s based on the RandomWatch schematics made by the people at RANDOM DATA.

Pressing the button shows the time for some seconds, then it shows the date and month.
